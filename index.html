LAUNCH A CAREER

Learning to code is one thing, becoming a developer is a whole different ball-game. The goal of our
traning at CodeDr Academy is to take you from hobbyist to professional software developer and be
the launchpad for your career.

CHOOSE A CAREER PATH

Full-Stack Engineering Career Path

A full-stack engineer can get a project done from start to finish. In this Career Path, you’ll
begin with the front-end, move on to the back-end, then learn to connect the two. By the
time you’re done, you’ll have the well-rounded skills needed to enter this in-demand job market.

Front-End Engineering Career Path

Front-end engineers work closely with designers to make websites beautiful, functional, and
fast. This Career Path will teach you not only the necessary languages and technologies, but
how to think like a front-end engineer, too. By the end, you’ll have the portfolio and interview
skills you need to start your new career.
